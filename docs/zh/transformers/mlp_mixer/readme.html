MLP 混音器：面向视觉的全 MLP 架构

这是论文 MLP-Mixer：视觉的全 MLP 架构的 PyTorch 实现。

本文将模型应用于视觉任务。该模型类似于变压器，其注意力层被应用于补丁上的 MLP（如果是自然语言处理任务，则为令牌）所取代。

我们的MLP Mixer的实现取代了变压器实现中的自我注意层。所以这只是几行代码，将张量转置以在序列维度上应用 MLP。

尽管本文将MLP Mixer应用于视觉任务，但我们在蒙版语言模型上进行了尝试。这里是实验代码。
