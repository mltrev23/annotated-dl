深度规范

这是《Dee p Net：将变形金刚缩放到 1000 层》一文中 DeepN orm 的 PyTorch 实现。

本文提出了一种通过新的归一化功能取代LayerNorm和权重初始化方案来稳定极深变压器的方法。这结合了 post-LayerNorm 的性能和 pre-LayerNorm 的稳定性。即使没有预热学习速率，带有 DeepNorms 的变形金刚也应该是稳定的。

本文首先表明，在稳定训练期间，图层输出（对于相同的输入）的变化会逐渐变化；如果不稳定，则在最初的训练步骤中会迅速变化。这种情况发生在将权重初始化为小值，以及在训练稳定的情况下进行学习速率预热时。他们使用保持对图层输出的更改较小的想法来推导新的归一化和权重初始化机制。

权重初始化

通常，权重通过 Xavier 或 Kaiming 初始化进行初始化。这篇论文β根据变压器的大小按常数缩放（设置增益）权重。

DeepNorm 建议缩放前馈网络中两个线性变换、值投影变换和注意力层的输出投影变换的权重。这些变换的权重按比例缩放（增益等于）β。

xl+1​=LN(αxl​+Gl​(xl​,θl​))

其中α是取决于变压器深度的常数，LN是层归一化，Gl​(xl​,θl​)是l-th 变压器子层（FFN 或注意）。

此函数用于替换 Post-LayerNorm。
