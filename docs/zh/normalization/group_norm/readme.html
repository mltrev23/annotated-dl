#

群组规范化

这是组规范化论文的 PyTorch 实现。

批量标准化适用于足够大的批量，但不适用于小批量，因为它在批次上进行标准化。由于设备的内存容量，无法使用大批量训练大型模型。

本白皮书介绍了群归一化，它将一组特征归一化为一组。这是基于这样的观察，即 SIFT 和 HO G等经典特征是按组划分的特征。本文建议将功能频道划分为几组，然后分别规范化每个组中的所有频道。
